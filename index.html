Como programar con el bloc de notas

Cosas importantes por definir

Aqui se mostrarán algunos algoritmos junto con su uso intentando ser lo mas objetivo posible a la hora de su explicacion

es decir, que cada uno puede ir creando la pagina a su gusto pero con el orden inicial que sera explicado proximamente.

"La mejor manera de aprender algo es practicando"

Para iniciar, las paginas web tienen un codigo que se ejecuta depende a como sea la orden dada, manteniendo *ctrl, intro y la letra I* se puede acceder a el codigo de la pagina web.

Se debe tener en cuenta siempre el orden en el que se escribe el codigo, pues un error puede hacer que no funcione algun algoritmo o todo el codigo

Orden del codigo

  • "html" pues es lo que hará que el dispositivo lo reconoca como pagina web.

  • "head" es en donde se colocará el titulo y la descripción de la pagina web.

  • viene "body" es donde estara la mayor parte del codigo logrando colocar desde textos hasta links de otra pagina web.

    • Recuerda que despues de terminar con cada parte del orden del codigo debes cerrarlo para asi lograr que todo salga perfecto, para eso se usa el /

    por ejemplo (< head >,< / head >)



    Programar una pagina web trae consigo el tema de decoración, como imagenes, links, color de fondo y de tablas, bordes, etc.

    (< ahref >)
    • Este comando sirve para colocar links en tu pagina web y se utliza escribiendo (ahref="link del sitio donde quiere que te envie") este comando no hace falta cerrarlo pues es solo un link

    (< table >)
  • Este comando se utiliza para crear una tabla y ademas de eso hay diversos comando mas para cambiar color del fondo, mostrar o no bordes y la letra que se usará.

    • (< table border >)
  • Este comando se utiliza de la siguiente manera: (table border="numero del 1 al 5") con eso se puede elegir si mostrar o no los bordes de la tabla.

    • (< width >)
    • Este comando se utiliza despues del anterior y de manera similar (table border="" width"10% al 100%") con esto se puede definir el tamaño de la tabla.

    (< bgcolor >)
  • Este comando se usa para definir el color del fondo de la tabla, se utiliza despues de "width" (table border="" width"" bgcolor="color en ingles") los colores deben ser en ingles gracias al lenguaje que se utiliza.

    • (< img src >)
  • Este comando se utiliza para colocar una imagen: (img scr="nombre del archivo con el .JGP, .PGN, etc.) ejemplo (< img scr="archivo para pagina.JGP >) este comando no es necesario cerrarlo pues es solo una imagen
